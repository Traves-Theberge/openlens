# Performance & Style Agent Research

Language-agnostic research for performance and style agents.

## Performance Agent
- N+1 detection across all ORMs (Django, SQLAlchemy, ActiveRecord, GORM, Prisma, Sequelize, Hibernate, EF)
- Algorithmic complexity estimation
- Memory/resource patterns (unbounded growth, leaks)
- I/O and network optimization
- Startup costs

## Style Agent
- Convention detection across 10+ language ecosystems
- Code smell detection (god functions, feature envy, primitive obsession)
- API design consistency
- Dead code detection
- Structural duplication

Both include: 5 decision trees, rationalization prevention, evidence requirements, common false positives.
