# Bugs Agent Research

Language-agnostic research for the bug detection agent. Covers null safety, error handling, resource leaks, concurrency, type system gaps, and contract violations across all programming languages.

## Key Sources
- skills.sh/obra/superpowers (systematic-debugging, verification-before-completion)
- Facebook Infer (null safety, resource leaks)
- Qodana/IntelliJ (inspections, type narrowing)
- Coverity (concurrency, error handling)

## Coverage
- Null patterns for 12+ languages with grep patterns
- Error handling anti-patterns (10 categories) with language-specific patterns
- Resource leak detection for 12+ resource types
- Concurrency bugs (race conditions, deadlocks, TOCTOU)
- Type system bypasses per language
- Contract violation detection
- 5 decision trees for ambiguous cases
- 15-item rationalization prevention table
- Evidence requirements (HIGH/MEDIUM/LOW)
- 22 common false positives
